--- layout: default title: Overview description: Summary of Metanorma features. ---

Easy semantic authoring.

With AsciiDoc-based syntax, you can write documents easily while maintaining the structure.

Structure and style validation.

Metanorma-based processors can validate your input against a formal schema for the document structure and style rules around content.

Flexible output.

Semantic input can be rendered HTML, PDF or other formats, and presented in way compliant Documents can be output in different languages, with language-specific templated text.

Multi-language support.

Documents can be output in different languages, with language-specific templated text.
