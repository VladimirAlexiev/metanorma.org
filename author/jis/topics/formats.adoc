== Metanorma for JIS output formats

The accepted format values of Metanorma-ISO output are as follows:

`html`:: HTML output
`doc`:: DOC output
`pdf`:: PDF output

[example]
.Generating output from a JIS document
====
[source,console]
----
metanorma -t jis -x html,doc,pdf test.adoc
----
====
