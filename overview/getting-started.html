Installing the Metanorma toolchain

Docker setup

  1. Pull the container: 
    docker pull ribose/metanorma
  2. Specify the :local-cache-only: AsciiDoc attribute in document header to speed up rendering
  3. To render the document into HTML, execute from within the directory containing the Metanorma document (replacing {my-document-file} with your actual document’s filename): 
    docker run -v "$(pwd)":/metanorma/ -w /metanorma ribose/metanorma metanorma -t iso -x html {my-document-file}

See metanorma-docker for more information and some shortcuts, including a pre-made Makefile.

All-in-one macOS setup

Run the following for an automatic install that would prepare a fresh system to work with Metanorma.

Note: it installs not just the Metanorma components but also the basics such as Homebrew and Sublime Text.

This should take less than 15 minutes depending on your machine and Internet connection: 

bash <(curl -s https://raw.githubusercontent.com/riboseinc/metanorma-macos-setup/master/metanorma-setup)
