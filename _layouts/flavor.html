On installing Metanorma toolchain, refer to installation guidelines.

  1. Install Metanorma

    Docker

    docker pull ribose/metanorma

    Homebrew

    brew tap riboseinc/metanorma

    Ruby gem

    gem install metanorma
  2. Write a document using Asciidoctor markup with {{ page.title }} extensions
    Asciidoctor quick reference {{ page.asciidoctor_flavor }} reference
  3. Specify the `:local-cache-only:` AsciiDoc attribute in document header to speed up rendering when using Docker
  4. Run Metanorma against your documents 
    docker run -v "$(pwd)":/metanorma/ -w /metanorma ribose/metanorma "metanorma {{ page.cli_flags }} [{{ layout.sample_document_name }}.{{ layout.sample_document_ext }}]"
